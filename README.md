A clean and responsive **Internship Listing Website** built using HTML and CSS.  
The page showcases internship programs (Web Dev, App Dev, Python, Java) in card format, each linking to a dedicated detail sub-page.

This project replicates a simple internship portal UI with a modern card-based layout and animated hover effects.

## Features

### Internship Cards  
Each internship program includes:
- Title  
- Duration  
- Location  
- Stipend  
- Start date  
- "View Details" button linking to a sub-page  

### Sub-pages Included
Each internship opens a minimal page confirming navigation works:
- `webdev.html`
- `appdev.html`
- `pythonpro.html`
- `javapro.html`

### Responsive Grid System  
The layout adapts to different screen sizes using CSS Grid with `auto-fit` & `minmax`.

### Modern UI Enhancements  
- Hover animations  
- Drop shadows  
- Rounded cards  
- Centered layout  
- Smooth transitions

## How to Run the Project

1. Clone the repo:
```bash
git clone https://github.com/venkata3204sai/voc-task-2
cd voc-task-2
```
2. Open the site in your browser:
```bash
open index.html
```
Or run a lightweight server:
```bash
python -m http.server 8000
# Visit → http://localhost:8000
```
